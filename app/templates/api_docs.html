Authentication

All API requests require the X-API-Key header.

X-API-Key: your-secret-api-key

Fraud Check API

GET /api/v1/fraud-check/{phone}

Checks delivery history across Steadfast, Pathao, CarryBee, and RedX.

Response Example:

{
  "phone": "01676225090",
  "summary": {
    "total_parcels": 150,
    "total_delivered": 80,
    "total_cancelled": 70,
    "steadfast_fraud_reports": 2,
    "overall_success_rate": 53.33
  },
  "couriers": [
    {
      "name": "Steadfast",
      "total": 15,
      "delivered": 7,
      "cancelled": 8,
      "comment": "fraud: 2",
      "status": "ok"
    }
    // ...other couriers
  ]
}

Parcel Create API

POST /api/v1/parcel/create

Creates a parcel in the chosen courier system.

Request Body Example:

{
  "courier": "carrybee",
  "business": "hellosquare",
  "recipient_name": "Jahid",
  "recipient_phone": "01676225090",
  "recipient_address": "Uttara, Dhaka-1230",
  "cod_amount": 1050,
  "city_id": 14,
  "zone_id": 151
}
