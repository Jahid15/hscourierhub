Authentication & Postman Setup

All external API requests to the system securely require the X-API-Key header to pass authentication. This securely blocks unauthorized systems from creating parcels or accessing your courier data.

How to test in Postman:

  1. Open a new Request in Postman and select your HTTP verb (e.g. POST).
  2. Paste your endpoint URL (e.g. https://hscourierhub.onrender.com/api/v1/parcel/create).
  3. Navigate to the Headers tab in Postman.
  4. Add a new Key named X-API-Key.
  5. Set the Value to your external API secret (matching your EXTERNAL_API_KEY environment variable).
X-API-Key: your-secret-api-key

Fraud Check API

GET /api/v1/fraud-check/{phone}

Checks delivery history across Steadfast, Pathao, CarryBee, and RedX.

Response Example:

{
  "phone": "01676225090",
  "summary": {
    "total_parcels": 150,
    "total_delivered": 80,
    "total_cancelled": 70,
    "steadfast_fraud_reports": 2,
    "overall_success_rate": 53.33
  },
  "couriers": [
    {
      "name": "Steadfast",
      "total": 15,
      "delivered": 7,
      "cancelled": 8,
      "comment": "fraud: 2",
      "status": "ok"
    }
    // ...other couriers
  ]
}

Parcel Create API

POST /api/v1/parcel/create

Pushes a new parcel entry autonomously into the requested courier pipeline and securely returns the Official Consignment and Merchant Order IDs for your system to track.

Request Body (JSON):

{
  "courier": "carrybee", 
  "business": "hellosquare",
  "recipient_name": "Jahid Hossain",
  "recipient_phone": "01676225090",
  "recipient_address": "Uttara, Dhaka-1230",
  "cod_amount": 1050,
  "city_id": 14,
  "zone_id": 151
}

Response Example:

{
  "success": true,
  "message": "Order created in carrybee",
  "consignment_id": "F0409JH4PQD",
  "merchant_order_id": "HS10982",
  "tracking_code": "0409JH4PQD",
  "courier": "carrybee"
}

Parcel Tracking Status API

GET /api/v1/parcels/{merchant_order_id}

Lookup the exact real-time telemetry updates your system collected via Automated Webhooks using the merchant_order_id returned during creation.

Response Example:

{
  "success": true,
  "data": {
    "merchant_order_id": "HS10982",
    "consignment_id": "F0409JH4PQD",
    "status": "Delivered",

  }
